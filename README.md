# Stock Sense

Stock Sense is an AI-powered market interpretation engine that translates raw price data into clear, structured insights.

Instead of predicting prices, it helps users understand what the market is doing.

---

## 🚀 Features

- 📊 Price context visualization
- 📈 Trend, Momentum, Structure, Bias analysis
- 🧠 Clear human-readable interpretation
- ⚡ Confidence score based on signal alignment
- 🪙 Crypto + Stocks support

---

## 🎯 Philosophy

Most tools show numbers.

Stock Sense explains meaning.

It does not predict the future —  
it helps you understand the present.

---

## 🧩 How it works

The system analyzes:

- Moving averages (SMA20/50)
- Short-term slope (momentum)
- RSI (relative strength)
- Realized volatility

These are combined into 4 axes:

- Trend
- Momentum
- Structure
- Bias

---

## ⚠️ Disclaimer

This is not financial advice.

The output is an interpretation of past data, not a prediction.
